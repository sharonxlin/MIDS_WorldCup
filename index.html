Inside #WorldCup2014

A Twitter guide to the World Cup

A project aimed at explaining and capturing the social essence of the World Cup. If you would like to learn more please contact Nikhita, Sharon or William. This is an open source project. If you are interested you can find the code HERE.







Become a World Cup Fan

The US has some of the most passionate sports fans in the world. Yet we do not represent when it comes to the world cup. We think this is due in part to the fact that the information on soccer is not as accessible as it is for other sports. And it certainly is not as accessible as it is in other countries. This project seeks to change that by creating engaging visualizations of World Cup related tweets.

Project Notes

This project was created by Nikhita Koul, Sharon Lin and William Lewis as a project for the W205-Storing and Retrieving Data and W209 Data Visualization and Communication classes. Both classes are part of the Masters in Data Science program at UC Berkeley .


Credits

The visualizations were designed with D3 and HighCharts. The search engine is powered by Apache Solr. The data comes from Twitter. The website was designed using Cool Kitten, a framework by Jalxob.The video was provided courtesy of SBnation.
